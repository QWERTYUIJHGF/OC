Work-case 1


1. Опишіть для чого використовують git, які основні дії та команди в ньому виконують.
Ініціалізація репозиторію - створення нового репозиторію Git для зберігання файлів проекту.
Додавання файлів до репозиторію - додавання файлів до Git-індексу, який вказує Git, що файли потрібно відстежувати.
Зафіксування змін - зафіксування змін, які були внесені у вихідний код, відбувається за допомогою команди commit. Комміти створюють нову версію проекту та дозволяють відслідковувати всі зміни, які 
були внесені.


Основні команди Git:


git init - ініціалізація репозиторію
git add - додавання файлів до Git-індексу
git commit - зафіксування змін
git branch - перегляд доступних гілок та створення нових
git merge - злиття гілок
git clone - клонування віддаленого репозиторію
git pull - отримання змін з віддаленого репозиторію
git push - надсилання змін до віддаленого репозиторію
